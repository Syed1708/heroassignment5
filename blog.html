Question-1: What are the different ways to select an element in the DOM?

we can select different ways select an element in the DOM like..

  • Select by ID document.getElementById("id")
  • Select by class name document.getElementsByClassName("class")
  • Select by tag name document.getElementsByTagName("tag")
  • Select by css class name but it select just first one document.querySelector("selector")
  • Select by css class name but it select all css selector document.querySelectorAll("selector")
  • etc

Question-2: What is the difference between innerHTML, innerText, and textContent ??

When we need to set html inside an element then we can use innerHTML

When we need to set text inside an element but ignore html or other text just show text then we can use innerText

When we need to set Plaintext inside an element if there any html tags not ignore but show as a plaintext then we can use textContent

Question-3: What is event delegation in the DOM?

This is one kind of technique where a parent element handles events for its child elements by taking advantage of event bubbling.

Question-4: What is event bubbling in the DOM?

when in a event call to a child element its automatically bubble up to his parent ! For example I have div with id parent and inside this div i have button id child if i set an event on child and when i click child btn then automatically bubble up to div parent

Question-5: How do you create, add, and remove elements using JavaScript?

Step 1 Create an element

let element = document.createElement("p");
element.innerText = "Hello World!";

Step 2 Add to dom

document.body.appendChild(element);

Step 3 Remove element

element.remove();
